2.8.76 CANx_SleepModeEnter Function

C code

void CANx_SleepModeEnter(void) // x - Instance of the CAN peripheral

Summary

Puts the CAN Peripheral in sleep mode (clock stop request).

Precondition

CANx_Initialize and CANx_MessageRAMConfigSet must have been called for the associated CAN instance.

Parameters

None.

Returns

None.

Example

CAN0_SleepModeEnter();

Remarks

None.
