2.100.23 SDMMC_READ_RESPONSE_REG Enum

C

```C typedef enum { // Read response register 0 SDMMC_READ_RESP_REG_0 = 0, // Read response register 1 SDMMC_READ_RESP_REG_1, // Read response register 2 SDMMC_READ_RESP_REG_2, // Read response register 3 SDMMC_READ_RESP_REG_3, // Read all the response registers SDMMC_READ_RESP_REG_ALL

} SDMMC_READ_RESPONSE_REG; ```

Summary

The enumeration lists the response registers

Description

The enumeration is used to identify the response register and is passed as an argument to the SDMMCx_ResponseRead() API to select the response to read.
