2.70.19 NVMCTRL_CacheInvalidate Function

C code

void NVMCTRL_CacheInvalidate ( void );

Summary

Invalidates all cache lines.

Description

This function invalidates all cache lines. This function is available if the library was generated with the Cache option (in MCC) enabled.

Precondition

Cache option in MCC should have been enabled.

Parameters

None.

Returns

None.

Example

// Invalidate all cache lines.
NVMCTRL_CacheInvalidate();

Remarks

None.
