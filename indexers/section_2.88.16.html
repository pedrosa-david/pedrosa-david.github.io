2.88.16 RAM_ECC_CallbackRegister Function

C code

void RAM_ECC_CallbackRegister (RAM_ECC_CALLBACK callback, uintptr_t context)

Summary

Registers a callback function for ECC events in RAM.

Description

This function allows the user to register a callback function that will be invoked during ECC events.

Precondition

None

Parameters

Parameters

Parameter Description
callback Pointer to the user-defined callback function.
context A user-defined value to be passed to the callback during
invocation.
Param Description
callback Pointer to the user-defined callback function.
context A user-defined value to be passed to the callback during
invocation.

Returns

None

RAM_ECC_CallbackRegister(EccCallback, (uintptr_t)NULL); ```
