2.10.39 CCP_TIMER_CALLBACK Typedef

C code

typedef void (*CCP_TIMER_CALLBACK) (uint32_t status, uintptr_t context);

Summary

Defines the function pointer data type and function signature for the ccp channel callback function.

Description

This data type defines the function pointer and function signature for the CCP channel callback function. The library will call back the client's function with this signature from the interrupt routine.
