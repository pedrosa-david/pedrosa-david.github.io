2.70.1 NVMCTRL_Initialize Function

C code

void NVMCTRL_Initialize( void )

Summary

Initializes given instance of the NVMCTRL peripheral.

Description

This function initializes the given instance of the NVMCTRL peripheral as configured by the user from within the MCC.

Precondition

None.

Parameters

None.

Returns

None.

Example

NVMCTRL_Initialize();

Remarks

This function should be called before calling any other NVMCTRL library function.
