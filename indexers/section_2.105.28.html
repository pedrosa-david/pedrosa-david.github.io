2.105.28 I2CSMBx_HostBufferRead Function

C

```C / x = I2C SMBUS peripheral instance number /

/ I2C SMBUS Host mode / uint32_t I2CSMBx_HostBufferRead(void* pBuffer) ```

Summary

Reads data from the internal buffer and into the application buffer

Description

Reads data from the internal buffer and into the application buffer

Precondition
