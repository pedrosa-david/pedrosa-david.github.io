2.28.2 DIVAS_DivSigned Function

C code

int32_t DIVAS_DivSigned(int32_t numerator, int32_t denominator)

Summary

This function uses the DIVAS peripheral to performs a signed 32-bit division.

Description

This function uses the DIVAS peripheral to perform a signed 32-bit division. The function takes a signed divisor and dividend and returns the quotient

Precondition

The peripheral should have been initialized by calling the DIVAS_Initialize() function once.

Parameters

Parameters

Parameter Description
numerator This should contain the signed dividend.
denominator This should contain the signed divisor.
Param Description
numerator This should contain the signed dividend.
denominator This should contain the signed divisor.

Returns

Returns quotient

Example

```C int32_t dividend; int32_t divisor; int32_t quotient;

dividend = 10000; divisor = -2000;

quotient = DIVAS_DivSigned(numerator, denominator) ```

Remarks

None.
