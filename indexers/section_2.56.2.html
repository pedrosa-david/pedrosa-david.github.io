2.56.2 IPCx_SetIRQHandler Function

C code

void IPCx_SetIRQHandler(ipc_irq_mask_t irq, IPC_CALLBACK handler, uintptr_t context);

Summary

Registers the callback function for interrupt handler

Description

This function sets the callback function which will be called from interrupt handler

Precondition

None.

Parameters

Parameters

Parameter Description
irq interrupt number
callback a pointer to the function to be called when value is
reached. Use NULL to Unregister the compare callback
context a pointer to user defined data to be used when the callback
function is called. NULL can be passed if no data needed
Param Description
irq interrupt number
callback a pointer to the function to be called when value is
reached. Use NULL to Unregister the compare callback
context a pointer to user defined data to be used when the callback
function is called. NULL can be passed if no data needed

Returns

None.
