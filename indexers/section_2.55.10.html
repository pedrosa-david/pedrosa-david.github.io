2.55.10 ICAP_CALLBACK Typedef

C code

typedef void (*ICAP_CALLBACK) (uintptr_t context)

Summary

Defines the function pointer data type and function signature for the ICAP callback function.

Description

This data type defines the function pointer and function signature for the ICAPx callback function. The library will call back the client's function with this signature from the interrupt routine.
