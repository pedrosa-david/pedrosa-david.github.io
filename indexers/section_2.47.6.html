2.47.6 GENERIC_TIMER_PeriodSet Function

C code

void GENERIC_TIMER_PeriodSet(uint64_t period);

Summary

Set the timer period value.

Description

This function sets the timer period value.

Precondition

None.

Parameters

Parameters

Parameter Description
period The period value of generic timer. For example, if the timer is
running at a frequency of 24 MHz and if the desired timeout is 10 milliseconds, then
the period value must be specified as 24MHz x (10/1000) = 240000 cycles
