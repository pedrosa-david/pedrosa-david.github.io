2.94.1 RTCC_Initialize Function

C code

void RTCC_Initialize(void)

Summary

Initialize the RTCC peripheral.

Description

This function initialize the RTCC peripheral as configured by the user from the MCC GUI.

Precondition

None.

Parameters

None.

Returns

None.

Example

RTCC_Initialize();

Remarks

This function should be called prior to calling any other RTCC peripheral library function.
