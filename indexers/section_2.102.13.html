2.102.13 SEFC_GpnvmBitRead Function

C code

__longramfunc__ uint32_t SEFC_GpnvmBitRead(void)

Summary

Reads GPNVM bits

Description

This function returns the GPNVM bits.

Precondition

None.

Parameters

None

Returns

32-bit GPNVM bit

Example

uint32_t gpnvmBits;
gpnvmBits = SEFC_GpnvmBitRead();

Remarks

It reads first 32 GPNVM bits, the following reads provide the next 32 GPNVM bits as long as it is meaningful. This function is executed from RAM
