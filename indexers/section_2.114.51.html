2.114.51 TCCx_Capture24bitCounterGet Function

C code

/* x = TCC instance number */
uint32_t TCCx_Capture24bitCounterGet( void );

Summary

Reads the timer current count value.

Description

This function reads the timer count value.

Precondition

TCCx_CaptureInitialize() function must have been called first.

Parameters

None.

Returns

Timer's current count value.

Example

uint32_t counter = 0;
TCC0_CaptureInitialize();
TCC0_CaptureStart();
counter = TCC0_Capture24bitCounterGet();

Remarks

The caller must know the number of significant bytes of timer. Period value is right-aligned.
