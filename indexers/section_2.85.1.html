2.85.1 QEIx_Initialize Function

C

```C / x = QEI instance number /

void QEIx_Initialize ( void ) ```

Summary

Initializes given instance of QEI peripheral in quadrature mode.

Description

This function initializes given instance of QEI peripheral in quadrature mode with options configured in MCC GUI.

Precondition

MCC GUI should be configured with the desired options.

Parameters

None.

Returns

None.

Example

QEI1_Initialize();

Remarks

This function must be called before any other QEI function is called. This function is normally called only be called once during system initialization.
