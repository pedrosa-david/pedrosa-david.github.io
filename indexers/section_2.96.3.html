2.96.3 RTT_Disable Function

C code

void RTT_Disable( struct tm *time )

Summary

Disables the real Time Timer

Description

This API is used to stop the real time timer.

Precondition

RTT should be running.

Parameters

None.

Returns

None.

Example

```C void My_callback(uintptr_t context) { //Interrupt recieved stop RTT RTT_Disable(); } int main(void) { RTT_Initialize();

//Enable callback for the periodic interrupt
RTT_CallbackRegister(My_callback, NULL);

//start RTT
RTT_Enable();

} ```
