2.90.4 RNG_WaitForTrngCnt Function

C code

void RNG_WaitForTrngCnt (void);

Summary

Waits for RNGCNT to reach the number of bits required for the PRNG seed.

Description

This function is used when TRNG generates seed for PRNG. It waits till the number of valid bits in RNGSEEDx registers reaches the Polynomial Length set via MCC GUI.

Precondition

RNG_Initialize must have been called before using this function.

Parameters

None.

Returns

None.

Example

RNG_WaitForTrngCnt();

Remarks

None.
