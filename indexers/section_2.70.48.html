2.70.48 NVMCTRL_DATA_FLASH_START_ADDRESS Macro

The start address of data flash varies based on device family. Refer to the generated header file for the actual start address value.

Summary

Defines the start address of NVMCTRL DATA_FLASH.

Description

This constant defines the start address of DATA_FLASH. It is recommended for the target application to use this constant in place hardcoded literals.
