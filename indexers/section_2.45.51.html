Summary

Data structure containing the SPI parameters which can be changed at run time

Description

This structure identifies the possible setup parameters for FLEXCOM SPI which can be changed dynamically if needed. An instance of this structure is passed as a parameter to the FLEXCOMx_SPI_TransferSetup API.
