2.88.7 RAM_ECC_Initialize Function

C code

void RAM_ECC_Initialize(void)

Summary

Enables ECC error detection interrupts for the RAM module.

Description

This function configures the RAM module to enable interrupts for both single-bit error correction (SERREN) and double-bit error detection (DERREN), ensuring error events are captured and handled.

Precondition

None

Parameters

None

Returns

Always returns true.

Example

RAM_ECC_Initialize();
