2.92.2 RSWDT_Clear Function

C code

void RSWDT_Clear( void )

Summary

Restart the RSWDT counter.

Description

This function is used to restart the RSWDT counter to avoid timeout reset/interrupt.

Precondition

None

Parameters

None.

Returns

None.

Example

//Application loop
while(1)
{
    RSWDT_Clear();
    //user code
}
