2.125.20 WDT_CallbackRegister Function

C code

void WDT_CallbackRegister( WDT_CALLBACK callback, uintptr_t context )

Summary

Allows application to register a callback with the WDT peripheral

Description

This function sets the pointer to a client/application function to be called "back" when the WDT expires. It also passes a context value (usually a pointer to a context structure) that is passed into the function when it is called. The specified callback function will be called from the peripheral interrupt context. The callback should be registered before starting the WDT timer.

Precondition

WDT_Initialize() function must have been called and the WDT must be started.

Parameters

Parameters

Parameter Description
callback_fn A pointer to a function with a calling signature
defined by the WDT_CALLBACK data type. Setting this to NULL disables
the callback feature.
context A value (usually a pointer) which is passed (unused)
into the function identified by the callback parameter
