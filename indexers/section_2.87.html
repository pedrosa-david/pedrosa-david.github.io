Name Description

device

device
QSPIx_MemoryRead Reads from the specified address of the serial flash
device
QSPIx_MemoryWrite Writes to the specified address of the serial flash
device
Name Description
QSPIx_Initialize Initializes given instance of the QSPI
peripheral
QSPIx_TransferSetup Setup QSPI operational parameters as desired by the
client.
QSPIx_WriteRead Write and Read data on QSPI peripheral.
QSPIx_Write Write data on QSPI peripheral.
QSPIx_Read Read data on QSPI peripheral.
QSPIx_IsBusy Returns transfer status of QSPI peripheral.
QSPIx_CallbackRegister Allows application to register callback with
PLIB.
