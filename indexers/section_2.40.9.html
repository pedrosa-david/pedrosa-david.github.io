2.40.9 EIC_PinDebounceStateGet Function

C code

bool EIC_PinDebounceStateGet ( EIC_PIN pin )

Summary

Gets the De-bounce state of the EIC Pin.

Description

This function gets the De-bounced state of the EIC Pin.

Precondition

EIC_Initialize() must have been called first for the associated instance. The debounce feature should have been enabled in MCC on the desired pin.

Parameters

Parameters

Parameter Description
pin EIC Pin number
Param Description
pin EIC Pin number

Returns

True, If the EIC Pin De-bounce state is set and False otherwise.

Example

if (true == EIC_PinDebounceStateGet(EIC_PIN_3))
{
    // The pin was debounced and the state of the pin is logic high.
}

Remarks

None.
